1. Перейти в профіль Github та на вкладку Repositories, натиснути New, налаштувати параметри та натиснути Create repository.
2. Для зберігання файлів.
3. Натиснути на кнопку з назвою поточної гілки, в рядок Find or create a branch... ввести назву гілки і натиснути Create branch <назва гілки> from <назва поточної гілки>
4. Створити Pull request і зробити Merge pull request.
5. Натиснути кнопку Code, і завантажити репозиторій кнопкою Download ZIP.
